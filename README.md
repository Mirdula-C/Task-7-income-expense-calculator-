# Income Expense Calculator

The **Income Expense Calculator** is a simple web application to help users manage their financial transactions by tracking incomes and expenses. It calculates the net balance in real-time and provides options to filter, edit, and delete transactions.

---

## Features

- Add transactions with a description, amount, and type (income or expense).
- View real-time net balance, total income, and total expenses.
- Filter transactions to view all, only income, or only expenses.
- Edit or delete existing transactions.
- Stores data in local storage for persistence.

---

## Technologies Used

- **HTML**: Structure of the web page.
- **CSS**: Custom styling with Tailwind CSS integration.
- **JavaScript**: Functionality for adding, editing, deleting, and filtering transactions.

---

## How to Use

1. Open `index.html` in any modern web browser.
2. Use the form to add transactions:
   - Enter a description.
   - Specify the amount.
   - Select whether it is income or expense.
3. Click **Add Entry** to save the transaction.
4. Use the filters to view specific transaction types.
5. Edit or delete transactions as needed.

---

## Project Files

- **`index.html`**: The main structure of the application.
- **`styles.css`**: Custom styling for the application.
- **`script.js`**: JavaScript logic for interactivity.
